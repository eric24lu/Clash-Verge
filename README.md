# Clash-Verge

自用的 Clash Verge Rev 全局扩展覆写配置与脚本集合。

## 功能概览
- 订阅节点按地区自动分组（便于策略选择）
- 策略按使用需求分组（流媒体 / AI / 社交 / 下载 等）
- 脚本：避免 DNS 泄漏
- 脚本：排除 Premium 节点（防止滥用或区分计费）

## 使用方式
1. 在 Clash Verge Rev 中导入本仓库的覆写配置文件（或复制相关片段）。
2. 将你的订阅链接放入订阅管理，开启自动分组。
3. 若使用脚本，请在配置中确保 script 扩展已开启。
4. 保存并应用。
